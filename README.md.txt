E-Commerce Sales Analysis using Pandas

Project Overview

This project analyzes an E-commerce sales dataset using **Python and Pandas** to extract meaningful business insights.

The goal is to demonstrate **data analysis skills**, including data cleaning, aggregation, business metric calculation, and visualization.

Dataset Features

The dataset contains information about:

- Order ID
- Order Date
- Customer Details
- City
- Product Information
- Category
- Quantity
- Price
- Discount
- Payment Method



Technologies Used

- Python
- Pandas
- Matplotlib (little)
- Jupyter Notebook


Key Business Metrics Calculated

Total Orders
Total number of unique orders placed.

Total Revenue
Revenue generated after applying discounts.

 Average Order Value
Average revenue per order.



Business Insights

Top Selling Products
Identifies products with the highest sales volume.

Revenue by Category
Shows which product categories generate the most revenue.

Top Cities by Revenue
Highlights cities contributing the most to total revenue.

Payment Method Analysis
Analyzes the most frequently used payment methods.

Monthly Revenue Trend
Examines revenue trends over time.

Top Customers
Identifies customers generating the highest revenue.



Example Visualization

Revenue distribution across product categories using a bar chart.
